Internet das Coisas


Sistema regulador de água com Arduino

Tema

Internet das Coisas: Sistema regulador de água com Arduino.

Delimitação do Tema

O projeto tem como proposta a criação de um protótipo de sistema regulador de água usando um sensor detector de nível de água que ao detectar um baixo nível de água abriria a válvula para encher o recipiente. O sistema regulador de água foi desenvolvido usando o Arduino e um site com informações sobre o projeto e um programa em Java para visualizar estatísticas do Arduino. Além disso, foi criado um servidor para ser hospedado o site. Este projeto foi desenvolvido pelos estudantes do Curso Técnico em Informática Michael Trage, Vanderson dos Santos e Ryan Zancanaro durante o período de Março à Julho de 2020.

Problema

Existe uma dificuldade regular a quantia de água de um recipiente na agropecuária que é bastante usado, poderia um Arduino ser capaz de tornar isso automático?

Hipóteses

Com um arduino dá para implementar várias funções num só equipamento.
Um sistema em arduino poderia facilitar o trabalho substituindo a mão de obra humana.
Não precisa de conexão à internet para funcionar.

Objetivo Geral

Criar um protótipo de sistema regulador de água em Arduíno.

Justificativa

A população mundial cresce mais e mais e desta forma é preciso que a produção de alimentos cresça ainda mais para poder alimentar toda essa população, e esse aumento na produção de alimentos se deve principalmente pela evolução na tecnologia no ramo da agropecuária, algumas das principais evoluções que contribuem para o melhor desempenho na agropecuária são a robótica, com máquinas autônomas por exemplo tem um grande rendimento na produção, a inteligência artificial, que é muito usada na análise de dados como monitorar a saúde e peso de um animal, sensores, que são muito importantes na coleta de informações, entre outros.

Já quando o uso de água é constante na área de agropecuária e não tem nenhum sistema automático para manter os reservatórios com água tem que fazer tudo automaticamente, o que se torna um pouco complicado, já que tem que checar os reservatórios e abrir o registro de água e então fechar, antes que vaze principalmente quando se tem vários recipientes que precisam sempre estar prontos para abastecer água a plantas ou animais, o projeto tem o objetivo de fazer um sistema que faria isso automaticamente e ainda teria um site com todas as informações do projeto e um sistema que ao conectar o arduino em um computador teria estatísticas sobre as últimas 24h.
